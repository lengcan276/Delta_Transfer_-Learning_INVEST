# Codex project instructions for INVEST manuscript

This repository contains a computational chemistry / machine-learning manuscript on inverted singlet-triplet gaps.

Important files:
- paper/main.tex
- data/processed/
- data/source/
- results/
- results/round1_eval/
- results/round2_eval/
- figures/
- scripts/

## Non-negotiable rules

1. Never write or edit numerical manuscript claims from memory.
2. All numerical claims must be traceable to:
   - results/canonical_metrics.json
   - figures/caption_data/*.json
   - generated tables in results/
3. Use manuscript-consistency before editing abstract, results, discussion, conclusions, captions, or tables.
4. Use stats-rigor-reviewer for p-values, LOO-CV, MAE/RMSE, accuracy, recall, coverage, bootstrap, conformal prediction, active learning, and baselines.
5. Use figure-auditor before changing or finalizing figures.
6. Use comp-chem-method-reviewer before making claims about ∆EST, INVEST, TADF, OLED, ADC(2), CC2, SCS-CC2, TDDFT, or basis-set reliability.
7. Use adversarial-reviewer before saying the manuscript is ready.
8. Do not use "verified", "robust", "significant", "high-confidence", "OLED emitter", or "multi-level verification" unless supported by audit results.

## Known manuscript risks

- Reconcile 15 round-1 queries versus any plotted 14-molecule subset.
- Reconcile 7/15 round-1 hits versus any 6/14 scaffold-panel display.
- Distinguish 33 post-round target, 14 deployment, 15 round-1 queries, and 35 validation molecules.
- Only 4 molecules have SCS-CC2 cross-checks.
- 52.1 meV LOO-CV MAE is larger than the 30 meV classification threshold.
- ADC(2)/def2-SVP assignments should be described as method-level candidates unless further validation exists.
